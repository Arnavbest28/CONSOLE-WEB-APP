ABOUT CONSOLE SCREEN

Console screen is a sreen where we can see the output of the code we have written. In 'Visual Studio' we have to install a option called 'Live Server'. You also have to check the browser in which console will open. After that when you will open any file, you will see 'go live' option at bottom right. If we click on it, Console screen will open on the choosen browser.
